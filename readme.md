# SkyTech Drones Professional Drone Services

## What’s included
- `index.html` — the full page (single-page site with sections)
- `styles.css` — all styling
- `script.js` — mobile menu + demo form success message
- `assets/` — placeholder SVG images you can replace

## Quick start (local)
Just open `index.html` in your browser.

## Replace the images
In `/assets`, swap the `.svg` files with real images:
- keep the same filenames (hero.svg, drone1.svg, etc.), OR
- update the `src="assets/..."` paths in `index.html`.
